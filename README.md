# 🌍 AI Language Translator

An AI-powered language translator developed using Python and Streamlit.

## Features

- 🌍 Translate into 20+ languages
- 🤖 Auto Language Detection
- 🎤 Voice Input
- 🔊 Text-to-Speech
- 📥 Download Translation
- 🕒 Translation History
- 📱 Responsive UI
- ✨ Premium Glassmorphism Design

## Technologies Used

- Python
- Streamlit
- Deep Translator
- Google Text-to-Speech
- SpeechRecognition

## Run

pip install -r requirements.txt

streamlit run app.py
